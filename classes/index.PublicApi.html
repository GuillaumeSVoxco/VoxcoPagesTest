setVariableMentions

  • setVariableMentions(mentions: string | MentionBuilderDelegate | string[], variableName: string, matrixIndex?: number): any

  • Set a variable's value.

    Example

     
    // String mention example
    Voxco.setVariableMentions("NewValue", "Q1", 1)
    // String[] mention example
    Voxco.setVariableMentions(["First mention value", "Second mention value"], "Q1", 1)
    // Mention builder example
    Voxco.setVariableMentions((mentions) => mentions.clear().add("New value"), "Q1", 1)

    See Also

    None

    Parameters

    • mentions: string | MentionBuilderDelegate | string[]

      New variable value.

    • variableName: string

      Variable being modified.

    • Optional matrixIndex: number

      Uses the current matrix index if not set.

    Returns any
